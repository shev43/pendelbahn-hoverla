# Pendelbahn Hoverla — 3D концепт на реальному рельєфі

Інтерактивна 3D-візуалізація **маятникової канатної дороги** (нім. *Pendelbahn*)
**Заросляк → Мала Говерла** на масиві **Чорногора** (Карпати), побудована на
**реальних геоданих**: рельєф SRTM, супутниковий знімок Esri, дорога з OpenStreetMap
і кадастрова ділянка для нижньої станції.

🔗 **Демо (GitHub Pages):** https://shev43.github.io/pendelbahn-hoverla/

> ⚠️ **Дисклеймер.** Це **художньо-концептуальна** візуалізація, а не проєкт
> будівництва. Гора Говерла — у межах Карпатського національного природного парку,
> капітальне будівництво канатних доріг тут не передбачене.

## Параметри траси (з реального DEM)

| Параметр | Значення |
|---|---|
| Нижня станція · Заросляк | 1259 м |
| Верхня станція · Мала Говерла | 1760 м |
| Перепад / горизонталь | 501 / 2343 м |
| Довжина тросу · ухил | ~2420 м · 21.4 % |
| Опори | стартова + долинна (2) |
| Кабіни | 2, рухаються зустрічно (маятник) |

Нижню станцію та стартову опору розміщено в кадастровій ділянці
`2611040300:21:008:0608` (0.89 га, рекреаційне призначення).

## Можливості

- Реальний рельєф (SRTM 30 м) з накладеним супутниковим ортофото (Esri, z16).
- Перемикач **🛰 Ортофото** — супутник ↔ відмивка рельєфу за висотою.
- Реальна дорога **Ворохта → Заросляк** (OpenStreetMap).
- Контур кадастрової ділянки, обидві опори, трос, дві кабіни.
- Вікно **📈 Профіль траси та дані** — поздовжній профіль рельєфу + параметри ліфта.
- Ракурси (станції, вершина, політ за кабіною), перемикачі шарів.

## Запуск локально

Статичний сайт на [Three.js](https://threejs.org/) (CDN, без збірки). Через ES-модулі
й `fetch` потрібен локальний HTTP-сервер (не `file://`):

```bash
npx serve .          # → http://localhost:3000
# або: python3 -m http.server 8000
```

## Структура

```
index.html            UI, importmap, підключення viewer.js
css/style.css         стилі
js/viewer.js          вся 3D-сцена (проєкція, рельєф, дорога, канатка, профіль)
js/data/elevation.json   висотна сітка (SRTM N48E024)
js/data/ortho.json       межі супутникової текстури
js/data/roads.json       дорога (OSM)
js/data/parcel.json      кадастрова ділянка (KMZ)
assets/ortho.jpg         супутниковий знімок (Esri World Imagery)
```
