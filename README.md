# Pendelbahn Hoverla — 3D концепт

Інтерактивна 3D-візуалізація **маятникової канатної дороги** (нім. *Pendelbahn*,
англ. *reversible aerial tramway*) на гору **Говерлу** (2061 м) — найвищу вершину
України, Чорногірський хребет, Карпати.

Концепт показує дві станції, проміжні опори, несучо-тяговий трос та **дві кабіни,
що рухаються зустрічно** (класична схема «маятника»: поки одна кабіна піднімається,
друга спускається, зрівноважуючи одна одну).

> ⚠️ **Дисклеймер.** Це **художньо-концептуальна** 3D-візуалізація, а не реальний
> проєкт будівництва. Гора Говерла розташована в межах Карпатського національного
> природного парку — капітальне будівництво канатних доріг тут не передбачене.
> Цифри (висоти, довжина, перепад) орієнтовні й слугують лише для наочності концепту.

## Параметри концепту

| Параметр                | Значення (концепт)        |
|-------------------------|---------------------------|
| Долинна станція         | ~1300 м н.р.м.            |
| Вершинна станція        | ~2000 м н.р.м. (плече)    |
| Вершина Говерли         | 2061 м н.р.м.             |
| Перепад висот           | ~700 м                    |
| Похила довжина траси    | ~2.3 км                   |
| Тип                     | Pendelbahn (2 кабіни)     |

## Запуск

Проєкт — статичний сайт на [Three.js](https://threejs.org/) (через CDN, без збірки).

Через будь-який локальний сервер (потрібен через ES-модулі):

```bash
# Python 3
python3 -m http.server 8000
# або Node
npx serve .
```

Потім відкрий <http://localhost:8000>.

> Просто відкрити `index.html` подвійним кліком (`file://`) не вийде —
> браузери блокують ES-модулі та текстури за протоколом `file://`.

## Керування

- **Ліва миша** — обертання · **Права/Shift** — панорама · **Колесо** — зум
- Кнопки зверху — швидкі ракурси (загальний вид, долинна / вершинна станція, політ за кабіною)
- Панель «Шари» — вмикання/вимикання опор, троса, підписів, рельєфу
